# OAS TOOLS CLI

## Setup
Install the command line tool with:
```sh
npm i -g @oas-tools/cli
```

Then execute the commands with the following syntax:

```sh
oas-tools <command>
```

Check the [documentation](https://oas-tools.github.io/docs/cli) for more information.
